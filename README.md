# ChirpStacktoDB
Sends Data from ChirpStack to MongoDB and pass the Data to Socket.io


Requirements 
--------------------------------------------------------
Nodejs with the following modules

socket.io-client 3.10.10

mqtt

mongodb

express

http

A MongoDB already Setup.
